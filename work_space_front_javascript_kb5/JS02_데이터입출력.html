데이터 입출력 방법

- JS를 통해 HTML이나 콘솔로 데이터를 입출력하는 방법

데이터 출력법

1. document.write() 활용

- 브라우저(html문서)로 출력하는 가장 기본적인 방법

2.window.alert('메세지')

- 브라우저 메세지창으로 출력하는 방법, window 생략 가능!
- 사용성 측면에서 좋지않아 주로 학습이나 디버깅 용로도 활용된다.

3. innerHTML, innerText ★★★★★

- html로 작성된 태그를 찾아 객체 내부에 html이나 text를 출력하는 방법
- 일반적으로 id를 찾아 태그를 조작하는 것이 표준적이다.

기본 문장

4. console.log()

- 개발자 도구 화면 콘솔로 출력하는 방법
- 일반 사용자에게는 보이지 않지만, 개발이 끝나면 정리하는 것이 바람직!
- 특징은 객체의 모든 상태를 상세히 볼수 있어 디버깅이 유용하다!

데이터 입력법

- html 태그중 input이 가능한 태그에서 사용자 입력값을 가져오는 방법
- 주로 서버로 전송하기 이전에 사용자 입력값을 체크하는 용도로 많이 활용

- 메세지 창을 통해 true/false으로 확인하는 방법, 사용성 이슈로 잘 쓰이지 않는다.

3. window.prompt()

- 사용자에게 텍스트를 입력받는 방법, 사용성 이슈로 잘 쓰이지 않는다. 2
