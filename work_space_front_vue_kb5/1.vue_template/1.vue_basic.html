Vue란?

- The Progressive JavaScript Framework (진보적인 JS 프레임워크)!
- SPA 프레임워크로 React의 라이벌, AngularJS의 비공식 후속 버전(개선 버전)
- Evan You로 고안됨, 구글 AngularJS를 가볍게 만들기 위해 시작된 프로젝트
- 문법상 특징은 html의 틀을 크게 깨지 않는 수준으로도 활용 가능(레거시에서 활용 가능!)
- 다만 최근 트렌드에 맞춰 Component 기반으로 설계를 구성함으로 react와 닮아가고 있다.
- 프레임워크로 다양한 도구(라우팅, CLI, 디버깅 도구)등을 지원한다.
- pure html base 보다 세심한 UI/UX를 구현할수 있도록 설계 되어 있음
- https://ko.vuejs.org/guide/introduction.html
- https://ko.vuejs.org/guide/extras/rendering-mechanism
