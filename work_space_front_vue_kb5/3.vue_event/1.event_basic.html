1. 이벤트 처리 - 인라인 핸들러 방식

- v-on을 활용하여 vue와 바인딩 된 객체 활용할수 있는 script를 생성하여 처리
- 코드 관리가 잘되지 않음으로 가급적 사용 안하길 권장!

금액 :

계좌 잔고 : {{balance}}


2. 이벤트 처리 - 메서드 핸들러

- @를 통해 vue 내부의 메서드 핸들러와 연결하는 방식
- 긴 이벤트 처리에도 적합하고 vue 코드에서 이벤트 처리 코드를 작성 가능
- 가급적 해당 방법을 사용하는 것을 권장, 사유는 코드 관리가 편해짐

금액 :

계좌 잔고 : {{balance}}
