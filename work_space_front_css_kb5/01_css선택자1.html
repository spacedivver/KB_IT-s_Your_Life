CSS 선택자란?

- 특정한 HTML 태그를 선택하여 스타일을 부여할때 활용
- 선택자 종류 : 태그, #id, .class, 후손, 자신, 이벤트 등등

1. 모든 선택자

- HTMl 문서 안에 모든 태그를 선택하는 기능

2. 태그 선택자 ★★

- HTML 문서안 특정 태그를 선택하는 방법
    ul 태그 영역입니다.
  • li 태그 영역입니다.

    • 3. 아이디 선택자(#) ★★★

    - HTML 태그의 속성값인 id를 통해 지정하는 방법
    - id와 style을 1:1로 맵핑 될때만 활용 가능
    - id는 일반적으로 JavaScript에서 사용되는 식별자로 가급적 css에서는 사용을 꺼림
    id1 영역입니다.
    id2 영역입니다.

    4. 클래스 선택자 ★★★★★

    - HTML 태그 속성중 class(그룹)로 지정하여 스타일을 적용하는 방법
    - class와 style을 1:n으로 매핑이 가능하여 가장 자주 활용되는 방법! (1:1도 가능!)
