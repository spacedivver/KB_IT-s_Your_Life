1. 텍스트 방향 지정하기

- 아랍권 사용자를 위해 글씨를 오왼순으로 정렬해주는 기능
왼쪽에서 오른쪽으로 가는 서구 스타일
오른쪽에서 왼쪽으로 가는 아랍 스타일
일타스랍아 는가 로으쪽왼 서에쪽른오

2. 글꼴 색상 지정하는 방법

- color 속성을 통해 글자색 지정 가능
- 방법1) 16진수 3byte or 4byte RGB값, ex) #FF00AA
- 방법2) rgb / rgba 함수 사용 ex) rgb(255, 0, 200);
- 방법3) 이미 표준화된 색상 텍스트로 정의 ex) red, blue
- VSCODE에서는 색상 플러그인을 통해 자유롭게 색상 조절 가능하다!
color1입니다. 마젠타 계열 투명색입니다.
color2입니다. 그린 계열입니다.
color3입니다. 블루입니다.
color4입니다. 삼성블루입니다.
