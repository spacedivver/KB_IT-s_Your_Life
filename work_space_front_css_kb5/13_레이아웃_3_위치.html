레이아웃 위치


1. position - 절대(absolute) / 상대(relative) 위치 ★

- 절대 위치 : 레이아웃 배치시 절대 위치(x,y)를 기반으로 태그를 배치시키는 방법
- 상대 위치 : 부모 혹은 다른 레이아웃을 기준으로 상대적인 위치를 정하는 방법

2. 고정위치(fixed) ★

- 드래그가 가능한 긴 웹페이지에서 드래그가 되어도 특정 위치에 고정하는 속성
고정위치1, 안보이는 이유 : 위치가 지정되지 않아서
top

3. visibility 속성 ★★

- 페이지에 특정 객체를 보이거나 보이지 않게 하는 속성
- 만일 보이지 않더라도 공간을 차지하는 옵션이 존재함
- 또한 보이지 않을 경우, 버튼이면 이벤트가 발생하지 않음
- display: none과 같이 외워두는게 좋은 속성
