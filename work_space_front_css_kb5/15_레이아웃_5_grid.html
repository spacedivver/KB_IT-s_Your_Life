0. display - grid 속성

- 참고 자료 : https://studiomeal.com/archives/533
- css3(2014년)에서 추가된 속성으로 2차원 영역에서 가로 세로로 레이아웃을 정하는 스타일
- container(외부 레이아웃)에 맞춰 알아서 크기를 변경하도록 설계가 가능하다.

grid 주요 속성

  • grid-template-columns : 그리드 레이아웃의 열의 갯수를 정의하고 열의 너비를 정하는 속성
  • grid-template-rows : 그리드 레이아웃의 세로의 갯수를 정의하고 세로의 너비를 정하는 속성

    2. 함수 사용하기

  • - repeat : 반복된 값을 넣는 함수, repeat(반복횟수, 반복할 인자)
    • - fr : 유연한 크기를 갖는 단위, 남는 자유 공간을 fr로 정해진 단위로 쪼개서 배치
    - ex) 100px 2fr 1fr : 100px를 할당하고 남는 공간중 2/3을 2fr로, 1/3을 1fr로 쪼개서 배치
