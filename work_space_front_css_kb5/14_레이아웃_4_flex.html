레이아웃 - flex


0. display - flex 속성 (flex=유연하다. 알아서 한다는 느낌)

- 참고 자료 : https://studiomeal.com/archives/197
- css3(2014년)에서 추가된 속성으로 1차원 영역으로 수평과 수직영역을 분리하는 속성
- 고전 layout은 고정 크기로 설계 하였지만, 모바일 등 유연한 반응형 레이아웃을 설계하기 위해 추가된 속성
- container(외부 레이아웃)에 맞춰 알아서 크기를 변경하도록 설계가 가능하다.
- ex) 6열 -> 가로 크기를 작게 변경 -> 3열 x 2단

flex container 주요 속성

  • flex-direction : 컨테이너가 Flex 항목을 쌓을 방향을 정의하는 속성
  • flex-wrap : 플렉스 항목의 가로가 래핑되어야 하는지 여부를 지정
  • flex-flow : flex-flow 모두 설정하기 위한 약식 속성
  • justify-content : Flex 항목을 가로로 정렬하는 데 사용
  • align-items : Flex 항목을 세로로 정렬하는 데 사용
  • align-self : 아이템의 교차축 정렬 정하는 속성
  • align-content : 세로 기준으로 item의 배치와 정렬을 결정하는 속성

  • 컨테이너가 Flex 항목을 쌓을 방향을 정의하는 속성

    • row-reverse : 오른쪽에서 왼쪽으로
    • column : 위에서 아래로
    • column-reverse : 아래에서 위로

    2. flex-wrap 주요 속성 활용

  • - 플렉스 항목의 가로가 래핑되어야 하는지 여부를 지정
    • nowrap : 한 줄에 표시
    • wrap : 여러 줄에 표시
    • wrap-reverse : 시작점과 끝점 바꿔 여러 줄에 표시
    • row wrap : 왼쪽에서 오른쪽, 여러 줄
    • row nowrap : 왼쪽에서 오른쪽, 한 줄
