여백 관련 옵션 ★★★★★


1. padding 옵션

- width / height로 정해진 구역(테두리)내에서 컨텐츠와의 간격을 의미
- 주요 옵션 : 전체, 상, 하, 좌 우를 따로 지정할수 있다.

0. 원본 padding 옵션
1. 전체 padding 30px 옵션
2. 상 : 20px / 좌 : 50px padding 옵션
3. 상하 : 50px / 좌우 20px padding 옵션 ※비추
4. 상 10px, 좌우 : 50px, 하 : 100px padding 옵션 ※비추
5. 상 10px, 우 : 50px, 하 : 100px, 좌 : 200px padding 옵션 ※비추
6. 실무 방법 : 전체 padding + 필요한 방향에 따로 지정
7. padding 마이너스 - 적용 안됨!
8. padding 패밀리로 적용

2. margin 옵션

- width / height로 정해진 구역(테두리)의 외부(밖) 컨텐츠간의 간격을 조정하는 옵션
- 주요 옵션 : 전체, 상, 하, 좌, 우를 따로 지정할수 있음.
- '-'(마이너스)가 사용 되는 옵션

0. 원본 margin
1. 전체 margin 30px
2. top 20px, left 100px margin
3. 상하 :50px, 좌우:100px margin
4. 상 : 50px, 좌우:100px, 하:150px margin
5. 상 : 10px, 우 : 50px, 하 : 100px 좌 :200px margin
6. 전체 margin 30px + top:100px
7. 상 -50px 좌 -10px

3. padding margin 복합사용

- 두 여백을 활용하여 복잡적인 사용이 일반적이다.
