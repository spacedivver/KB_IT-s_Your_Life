2. 기본적인 레이아웃 배치(고전=block, inline, inline-block, none) ★★★★★

- display 속성 : lnline / block / inline-block / flex / grid 값을 통해 레이아웃을 배치할때 활용되는 중요 속성!★★★★★
- block, inline, inline-block, none를 고전적으로 많이 사용됬고, flex, grid는 css3 나오면서 추가된 문장(IE에서 지원 안했음!)
- Tip : block요소 = div태그 / inline요소 = span태그로 감싸서 활용 할 것!

display 주요 속성(고전)

  • block : 블록(한줄 또는 영역)으로 지정하는 요소로 세로 레이아웃을 만들때 사용
  • inline : line(한줄)에서 일부 태그를 감쌀때 사용하는 태그로 영역을 가질수 없음
  • inline-block : inline 요소를 옆으로 block 처럼 쌓아 가로 영역을 확보할때 사용
  • none : 태그 요소를 페이지에서 안 보이도록 변경하는 속성(가장 표준적인 문법!!) ★★★★★
