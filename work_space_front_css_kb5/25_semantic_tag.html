Semantic Tag




시멘틱 태그의 종류

1. header 태그

- page 또는 본문(body)의 header를 표시할때 사용

2. nav 태그

- 네비게이션 내용을 표현, 위치가 자유롭다.

3. aside 태그

- 부가적인 내용, 목록, 광고, 날씨정보

4. article 태그

- 본문 내용, 내용이 독립적이어야한다.

5. section 태그

- 연관된 컨텐츠를 그룹필할때 사용

- 내부 내용은 div로 분리하는 것이 일반적

6. footer 태그

- 사이트 하단의 내용, 본문 하단의 내용으로 구성

- 저작권, 회사 소개, 전화번호, 연락처
