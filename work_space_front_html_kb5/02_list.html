목록 관련 태그


1. ul(unordered list) - li(list) 태그 : 순서가 필요 없는 목록 태그 ★★★★★

-> 현업에서는 실제 메뉴를 표현을 할 때 주로 ul-li로 표현한다.

ul 태그 : li를 감싸는 외부 태그를 순서가 필요 없을때 감쌈
li 태그 : 실제 컨텐츠를 담는 목록(list) 태그
  • 리스트1
  • 리스트2
  • 리스트3
  • 리스트4

2. ol - li : 순서가 있는 목록 태그

ol 태그 : li를 감싸는 외부 태그로 순서가 있을때 활용
li 태그 : 실제 컨텐츠를 담는 목록(list) 태그
