미디어 관련 태그

이미지 태그 (img)

- 웹페이지에서 사진이나 이미지를 삽입할때 사용하는 태그, 인라인 태그
- 링크(주소)를 통해 이미지 파일을 별도로 가져와서 표시하는 기능

이미지 첨부1 - 다른 사이트에서 이미지 가져오기

주의점 : 'http' 'https'를 포함한 주소지가 필요!


이미지 첨부2 - 내부 파일(서버)로 부터 가져오는 방법 - 상대경로로 가져오는 방법★★★★★

주의점1 : /로 시작하면 안된다!
주의점2 : ./로 시작하면 되는데, 일반적으로 생략하는 표현을 더 많이 쓴다.



이미지 첨부3 - 절대경로로 가져오는 방법, 절대 쓰지말것!!! -> 서버일때는 안보인다.


이미지 첨부4 - alt 이미지가 안보일때의 text문구 추가하기



이미지 첨부5 - 이미지 크기 조정하기

주의점 : width, height를 임의로 지정하는 경우 사이즈가 깨질수 있다!
비율을 유지하기 위해선, width, height중 하나만 걸어준다.


map 태그

- 이미지 구역을 분리하여 링크를 걸어주는 태그

map : 이미지 구역을 생성하는 태그

area : map 내부에서 이미지 구역을 지정하는 태그


오디오 태그 (audio)

- 웹브라우저에서 별도의 플러그인 없이 음악을 재생시켜주는 태그

비디오 태그(video)

- 웹 브라우저에서 플러그인 없이 미디어를 재생시켜주는 태그

동영상 재생 - 컨트롤 표시 (controls)

동영상 재생 - 재생 전 출력할 이미지 지정(poster), 크기 지정
