Container

- 컨테이너는 부트스트랩의 가장 기본적인 레이아웃 요소로 그리드, 정렬, 공간 분리시 활용
- 반응형 UI 자동으로 적용되어 있음으로 해상도 별로 크기 조절이 가능하다.

기본형 컨테이너이고, small 사이즈와 동일하다.
