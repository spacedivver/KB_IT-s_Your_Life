Bootstrap이란?

- 기존 웹의 문제점 : 표준이 없다보니 웹구성 스타일이 제각각이었다.
- 스타일을 정립하려고 해도 코드 재사용과 디자이너/개발자의 협업시에 항상 불협화음 발생했다.
- 이러한 스타일을 정리하기 위해 트위터에서 탄생하게 된것이 Bootstrap
- 특징1. CSS의 class속성을 통해 미리 정의된 특정 CSS 템플릿 디자인을 적용 가능
- 특징2. class의 이름은 단축 표현되어 있어 어느정도 암기가 필요.
- 특징3. 정의된 class속성간 중첩을 통해 스타일을 중첩 시킬수 있음
