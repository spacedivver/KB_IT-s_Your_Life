속성 선택자

요소[속성] : 특정 속성을 가진 객체 선택

요소[속성 = 값] : 속성 안의 값이 특정 값과 같은 경우 ★★★★★

요소[속성 ~= 단어] : 속성 안의 값이 특정 단어를 포함하는데, 완벽하게 일치하는 객체 선택

요소[속성 ^= 단어] : 속성 안의 값이 특정 단어로 시작하는 객체 선택

요소[속성 $= 단어] : 속성 안의 값이 특정 단어로 끝나는 객체 선택

요소[속성 *= 값] : 속성 안의 값이 특정 값을 포함하는 객체 선택★★
